# Migrating to v0.30

Breaking changes from the DocCov simplification.

## Removed CLI Flags

| Flag | Replacement |
|------|-------------|
| `--tier` | Removed, was unused |
| `--update-snapshot` | Use `--write` on `spec` command |

## Removed Commands

- `doccov info` - Use `doccov check --verbose` instead

## Removed Report Formats

HTML/GitHub/PR-comment reports moved to `@doccov/action`. CLI now outputs:
- `text` (default)
- `json`

## Config Changes

```diff
// doccov.config.ts
export default {
  // Removed
- tier: "pro",
- snapshots: { update: true },

  // Still supported
  packages: [...],
  minCoverage: 80,
}
```

## Package Moves

SaaS packages moved from `packages/` to `apps/platform/`:
- `@doccov/db`
- `@doccov/auth`
- `@doccov/api-shared`
- `@doccov/sandbox`
- `@openpkg-ts/ui` (internal)

These are internal and not published to npm.
